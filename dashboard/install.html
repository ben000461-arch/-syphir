Open Chrome Extensions

Click the button below — it opens the Chrome extensions page where you'll install Syphir.

⚠ Enable Developer Mode
On the extensions page, toggle Developer mode ON in the top right corner. This allows installing verified extensions like Syphir.
✓ URL Copied!
1. Open a new Chrome tab
2. Paste in the address bar: chrome://extensions
3. Press Enter
4. Toggle Developer mode ON (top right)
5. Click Load unpacked → select the syphir-extension folder

Load the Extension

Follow these steps on the Chrome extensions page.

1
Unzip the downloaded file
Find syphir-extension.zip in your Downloads. Double click to unzip — you'll get a folder called extension.
2
Click "Load unpacked"
On the Chrome extensions page click Load unpacked in the top left.
3
Select the extension folder
In the file picker select the extension folder — not the zip file, the folder inside it.
✓
You're done
Syphir Shield will appear in your extensions. Click the puzzle piece 🧩 in the Chrome toolbar and pin it.

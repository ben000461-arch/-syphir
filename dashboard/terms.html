2. Description of Service

Syphir is a browser extension and web dashboard that monitors AI tool usage within an organization to detect and log instances where sensitive data (such as personally identifiable information, financial data, or confidential records) is submitted to AI tools including ChatGPT, Claude, Gemini, and others.

The Service provides administrators with an incident dashboard, alerting, and reporting capabilities. Employees are notified in real time when sensitive data is detected before submission.

4. Data Collection

Syphir is designed to minimize data collection. The extension scans prompt text locally in the browser and logs only metadata when sensitive data is detected:

  • The type of sensitive data detected (e.g., "SSN", "Credit Card") — not the actual value
  • A masked snippet of the detected value (e.g., "12****89")
  • The AI tool being used, the timestamp, and a risk level classification
  • The employee's email address (as provided during setup)

Full prompt text is never transmitted to or stored on Syphir servers. All scanning occurs client-side in the browser extension.

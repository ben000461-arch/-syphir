How it works
Three steps. Zero friction.

Deploys in minutes with no IT department required. Employees install once, then Syphir works silently in the background.

01

Business gets a license key

Your unique org key (SYP-XXXX) is your identity. It links every employee install back to your dashboard.

02

Share one link with employees

Your dashboard generates a pre-linked install URL. Employees click it — your key is already baked in.

03

All incidents flow to your dashboard

Every detection from every employee shows up only in your org's dashboard — isolated from all other businesses.

Employee onboarding
One link. Auto-linked to your org.

No employee ever has to think about keys or configuration. The install link does everything.

Your org's install link looks like:

https://syphir.vercel.app/install.html?key=SYP-YOUR-ORG-KEY
1

Admin copies install link from dashboard

Found in Settings → Team Members → Manual Install Link. The ?key= param is pre-filled with your org key.

2

Share via email, Slack, or text

Employee clicks the link. The install page reads the key from the URL and bakes it into the extension config automatically.

3

Employee installs in 60 seconds

No account needed, no key to type. The extension activates and immediately starts protecting that employee's AI sessions.

4

Dashboard shows everything — scoped to your org

Business A only ever sees Business A's incidents. Complete isolation. Your employees' activity never mixes with another org.

Features
Enterprise protection.
Startup simplicity.

40+ PII entity types

SSNs, credit cards, medical records, API keys, DOBs, EINs, IBAN, and more — all detected with pattern-based precision.

Real-time interception

Scans happen before the prompt is sent. No data leaves the browser until your employee confirms the risk.

Per-org incident dashboard

Each business sees only their own incidents. Complete data isolation — Dental Office A never sees Law Firm B's data.

Zero sensitive data stored

Syphir only logs metadata — entity type, risk score, timestamp. The actual sensitive text never touches our servers.

Works on all AI tools

ChatGPT, Claude, Grok, Gemini, Copilot, Perplexity — and any new tool added with one config change.

Multi-org management

Manage multiple practice locations or clients from one admin account with full per-org isolation.

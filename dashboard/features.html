Features

Everything you need | to hold the line.

Built for small businesses that handle sensitive data daily — without the complexity or price tag of enterprise solutions.

// pii

40+ sensitive data types

SSNs, credit cards, medical records, API keys, DOBs, EINs, IBAN numbers, driver's licenses, and more — all detected with precision before a prompt ever sends.

// realtime

Real-time interception

Scans happen before the prompt is submitted. No data leaves the browser until the employee confirms the risk is intentional.

// dashboard

Per-org incident dashboard

Each business sees only its own incidents. Complete isolation — your data never mixes with another organization's.

// privacy

Zero sensitive data stored

co|op only logs metadata — entity type, risk score, timestamp. The actual sensitive text never touches our servers.

// coverage

Works on all major AI tools

ChatGPT, Claude, Grok, Gemini, Copilot, Perplexity — and any new tool can be added to the allowlist with one config change.

// onboarding

One-link employee onboarding

Share one pre-linked URL with your team. They install in 60 seconds with no account, no key to type, no configuration.

// policies

Custom detection policies

Add your own keyword rules and sensitivity thresholds on top of the built-in detectors, tuned to your industry.

// feed

Live incident feed

Watch detections stream in as they happen. Filter by risk, employee, AI tool, or time window.

// audit

CSV export & audit logs

Export any incident history for compliance, board reporting, or your own records — one click, clean data.
